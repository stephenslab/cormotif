Simulation

We set unit number \(n = 5000\), and study number \(R = 4\). We set every observation has the same standard error, \(s_{jr}=1\). That is \(\hat\beta_{jr}|\beta_{jr}\sim N(\beta_{jr};0,1)\). The 5000 units come from 4 patterns (\(K=4\)): 4500 units have zero effects in all four studies, that is \(\beta_{jr}=0\), for \(r = 1,2,3,4\); 200 units have effect \(\beta_{jr}\sim N(0,10^2)\) for \(r =1, 2\) and \(\beta_{jr}=0\) for \(r =3, 4\); 200 units have effect \(\beta_{jr}\sim N(0,10^2)\) for \(r =2, 3\) and \(\beta_{jr}=0\) for \(r =1, 4\); 100 unitss have effect \(\beta_{jr}\sim N(0,10^2)\) for \(r =1, 2,3,4\).

Known \(K\)

First assume we know \(K=4\) and fit the model.

source('~/Git/cormotif/code/function.R')
K = 4
D = 4
G = 5000
sigma2 = c(100,100,100,100)
pi0 = c(0.02,0.04,0.04,0.9)
q0 = matrix(0,nrow=K,ncol=D)
q0[1,] = c(1,1,1,1)
q0[2,] = c(1,1,0,0)
q0[3,] = c(0,1,1,0)
q0[4,] = c(0,0,0,0)
X = matrix(0,nrow=G,ncol=D)
rows = rep(1:K,times=pi0*G)
A = q0[rows,]
set.seed(111)
for(g in 1:G){
  for(d in 1:D){
    if(A[g,d]==1){
      X[g,d] = rnorm(1,0,sqrt(1+sigma2[d]))
    } else{
      X[g,d] = rnorm(1,0,1)
    }
  }
}
betahat=X
sebetahat=matrix(1,nrow=G,ncol=D)
fit1 = generic.cormotif(betahat,sebetahat,K=4,mess=FALSE)

Discussion
